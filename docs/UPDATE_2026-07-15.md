# Update: Token Dashboard Layout

## What changed

- Added today's Codex token total, input tokens, cached input tokens, and output tokens to the BLE payload.
- Added a GPT-5.6 API-equivalent cost estimate in USD.
- Added 24 hourly token-usage bars to the home screen.
- Preserved the BLE connection state, Codex status, plan, seconds-precision clock, room temperature, humidity, and battery indicator.
- Kept the stock page, stock refresh flow, page button, and 10-second BLE refresh interval unchanged.
- Tuned the status-row typography so the room sensor text stays inside the display.
- Reduced and repositioned the weekly percentage text for a cleaner progress-card fit.

## Why

The previous home screen could clip the room sensor text after the status fonts were enlarged. The new layout uses the available 400x300 landscape area more carefully while making the token activity easier to read at a glance.

## Validation

- Arduino CLI compilation passed for the ESP32-S3 target.
- Firmware was flashed to COM5 and verified by esptool.
- BLE reconnected after reboot and returned `OK updated`.
- The bridge continues to refresh Codex and stock data every 10 seconds.

## Pricing note

The USD value is an API-equivalent estimate based on GPT-5.6 Sol rates, not a subscription invoice or a statement of actual account billing.
